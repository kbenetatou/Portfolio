Behavior Pattern Detection with Viterbi

While my focus lies in user-centered design, my background in Computer Science often complements how I approach complex problems. One example is a Python-based tool I created to analyze system message bursts and detect behavioral states using the Viterbi algorithm.

  • Reads time-series data from input files to model system activity over time
  • Identifies transitions between “high” and “low” activity states
  • Uses algorithmic pattern recognition to make sense of raw message emissions
  • Includes customizable flags for detailed output and debugging

Though not user-facing, this project reflects my interest in data-driven reasoning and behavioral modeling — useful in UX for personalization, segmentation, or designing smarter interactions.

Recursive Tiling Algorithm in Python

This project explores a classic divide-and-conquer algorithm through a recursive implementation of the tromino tiling problem. The program accepts a board size of 2n×2n and visualizes how the entire board can be tiled using L-shaped trominoes — leaving one “missing” square as a hole.

  • Recursive approach that breaks the board into quadrants
  • Uses ASCII letters to simulate colored tiles (G, B, R, X)
  • Demonstrates algorithmic thinking and problem decomposition
  • Visual output builds intuition for geometric algorithms

While the project is more mathematical in nature, it reflects the logical structuring and recursion skills that continue to shape how I think about flows and patterns — both in interfaces and code.
